# EventMaster ‚Äî Group Project
EventMaster is a **dynamic event management website** developed as a **group project by a team of three students**.  
The project demonstrates server-side web development, database integration, and clean application logic using **PHP and MySQL**.

## Team Information

- Project Type:** Group Project  
- Team Size:** 3 members  
- **Development Style:** Collaborative planning, development, and testing  

## Project Overview

EventMaster allows users to interact with a dynamic web application where they can:
- Register and log in securely
- Log out and manage sessions
- Browse event pages
- Book events and manage bookings
- Store and update data using a relational database

The website is designed to be visually clean, easy to navigate, and logically structured.

## Key Features
  - User Authentication
  - User registration
  - Login and logout using PHP sessions
  - Input validation for authentication forms

  - Event Booking System
  - Create and manage event bookings
  - Booking information stored in database tables
  - Ability to update booking records

  - Database & Logic
  - MySQL database with tables for users, events, bookings, and updates
  - Structured SQL queries for inserting, retrieving, and updating data
  - Validation to ensure data integrity and prevent invalid entries

- **Dynamic Website**
  - PHP used to handle form submissions and page logic
  - Dynamic content rendered based on database data and user actions
